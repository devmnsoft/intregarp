# Roteiro Demo v1.2 — Jornada do Cliente

1. Login como administrador.
2. Ver barra de onboarding.
3. Continuar primeiros passos.
4. Criar setor.
5. Criar usuário.
6. Instalar template.
7. Criar primeiro cliente.
8. Criar primeiro produto.
9. Criar pedido.
10. Ver “O que fazer agora”.
11. Resolver tarefa recomendada.
12. Abrir ajuda contextual em uma tela.
13. Ver estado vazio inteligente.
14. Perguntar à IA: “O que eu faço agora?”
15. Ver resposta guiada.
16. Ver score de adoção.
17. Entrar como usuário de campo.
18. Ver próxima tarefa no mobile.
19. Concluir tarefa com instruções.
