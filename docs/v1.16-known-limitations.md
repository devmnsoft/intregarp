# IntegraRP v1.16 — Limitações Conhecidas

- Ambiente de execução sem SDK .NET, Docker e PostgreSQL impediu validação de build/test/migrations.
- Mobile não possui script `npm test`.
- `npm audit --omit=dev` retornou 403 do registry.
- Autenticação real persistida ainda não foi concluída; endpoints mobile de login/refresh retornam 501.
- Flow, Project Central, Studio, BI e partes de Billing/Connect ainda precisam substituir implementações in-memory/sandbox por PostgreSQL antes de homologação plena.
