## Sumário executivo
Status geral: **Parcial/Risco**. O repositório contém fundação ampla, migrations até v1.5 e script único, mas o ambiente Codex não possui SDK .NET nem Docker. A v1.6 adiciona evidência automatizável por CI, manifesto de migrations, scripts Windows e contratos de smoke test.

- **OK:** `database/scriptcompleto.sql` existe, usa schema `integrarp`, contém `schema_migrations`, `fn_set_atualizado_em`, índices, triggers e views.
