# IntegraRP foundation

Documento da fundação IntegraRP. Consulte o README para comandos operacionais e a implementação em `src/`, `database/` e `scripts/` para detalhes executáveis.

## Diretrizes

- Usar schema único `integrarp`.
- Preservar Clean Architecture.
- Manter Windows local e Docker como caminhos suportados.
- Não colocar segredos reais no repositório.
- Manter IA governada por ferramentas internas autorizadas.
