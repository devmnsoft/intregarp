# IntegraRP v1.16 — Guia de Deploy

1. Configurar `INTEGRARP_JWT_SECRET` ou `Jwt:Secret` fora do repositório com no mínimo 32 caracteres.
2. Em Production, configurar `Cors:AllowedOrigins` com as origens autorizadas.
3. Não publicar Swagger em Production; o código só habilita Swagger em Development.
4. Executar `database/scriptcompleto.sql` duas vezes em banco vazio para validar idempotência.
5. Executar o Migration Runner antes de liberar tráfego.
6. Validar health checks `/health`, `/api/health/live` e `/api/health/ready`.
