# Roteiro de demonstração - Integra Studio Avançado

1. Subir API, Web, Worker e banco.
2. Fazer login no Web.
3. Abrir Studio em `/studio`.
4. Criar módulo “Controle de Avarias”.
5. Criar campos Cliente, Produto, Lote, Quantidade, Tipo de avaria, Foto, Descrição, Responsável, Status, Data da ocorrência e Custo estimado.
6. Criar ações Enviar para análise, Aprovar, Reprovar, Gerar tarefa para estoque, Gerar tarefa para financeiro e Iniciar processo de devolução.
7. Configurar BPMN “Tratamento de Avaria e Devolução”.
8. Configurar KPIs Total de avarias, Avarias abertas, Tempo médio de resolução e Custo estimado total.
9. Configurar catálogo semântico com perguntas permitidas e campos sensíveis bloqueados.
10. Publicar módulo.
11. Acessar módulo pelo menu em `/dynamic/controle_avarias`.
12. Criar registro de avaria.
13. Executar ação “Enviar para análise”.
14. Ver tarefa/Flow iniciado.
15. Consultar KPI do módulo.
16. Perguntar à IA sobre avarias, validando permissões.
17. Ver auditoria recente.
