## Evidência complementar deste ajuste

- Os `.csproj` minificados de API, Application, Infrastructure, Web e Worker foram normalizados para facilitar `dotnet format` e revisão humana.
- `Microsoft.Extensions.Hosting.Abstractions` do Worker foi alinhado para a linha 8.x compatível com `net8.0`, removendo o pacote 10.x identificado no diagnóstico.
- O refresh token agora consulta a sessão persistida, bloqueia renovação de sessão revogada e revoga toda a família ao detectar reutilização de token já consumido.
