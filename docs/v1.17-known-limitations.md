# v1.17 — Identidade, tarefas e homologação executável

## Entregue neste incremento
- Autenticação JWT persistida em PostgreSQL por tenant, e-mail e senha, usando o hasher do ASP.NET Core, sessões e refresh token armazenado como hash.
- Endpoints de autenticação corporativa: login, refresh, logout, me, change-password, forgot-password e reset-password.
- Tarefas com endpoints principais para assumir, iniciar, salvar formulário, salvar checklist, comentar, evidenciar, concluir e cancelar.
- Migration 0025_v117_auth_tasks_mobile_persistence.sql, manifest e scriptcompleto atualizados.

## Limitações honestas
- O ambiente local não possui SDK dotnet; build, testes e format precisam ser confirmados no CI ou em máquina com .NET 8.
- Web MVC completo, administração visual final, storage binário endurecido e worker de conflito offline permanecem para próximo incremento.
