# Roteiro de demonstração — Sprint 8 Mobile + IA

1. Subir API, Web, Worker e banco.
2. Fazer login no Web.
3. Criar ou abrir processo/pedido demo.
4. Gerar tarefa para usuário de campo.
5. Abrir app mobile.
6. Login no app.
7. Abrir “Minha fila”.
8. Assumir tarefa.
9. Preencher checklist.
10. Adicionar foto/evidência.
11. Capturar GPS.
12. Capturar assinatura.
13. Concluir tarefa.
14. Ver atualização no Web.
15. Abrir Console IA.
16. Perguntar status do pedido.
17. Perguntar KPI permitido.
18. Tentar pergunta sem permissão e ver fallback.
19. Abrir auditoria da IA.
20. Ver tarefa humana gerada por fallback.
