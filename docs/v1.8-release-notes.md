# v1.8-release-notes

## V1.8 status

This document was created for the V1.8 functional-product track.

## Current deliverable

- Real domain tables were added to `database/scriptcompleto.sql` and migration `database/migrations/0020_v18_funcionalidade_real_produto.sql`.
- Domain PostgreSQL repository classes were added under `src/IntegraRP.Infrastructure/Repositories/Postgres/**` and point to concrete domain tables.
- The legacy `integrarp.v15_operational_object` table remains documented as readiness metadata only, not as CRUD storage for real modules.

## Known limitation

The container used for this implementation does not include the .NET SDK, so build and tests could not be executed locally here. The attempted command failed with `dotnet: command not found`.
