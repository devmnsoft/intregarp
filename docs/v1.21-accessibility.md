# v1.21 — Accessibility

Documento de evidência e arquitetura para a v1.21.

## Estado implementado

- Shell MVC preservado com Bootstrap 5 local, JavaScript sem framework pesado e assets versionados.
- Design tokens, CSS base, shell, componentes, utilitários e acessibilidade separados.
- Navegação agrupada por contexto e preparada para filtro por ambiente/perfil.
- Estados vazios evitam dados fictícios em Production.

## Limitações honestas

- O ambiente local não possui SDK .NET nem `psql`; validação completa deve ocorrer no CI ou em estação homologada com PostgreSQL 16.
- Screenshots e Playwright foram estruturados no workflow como etapa de homologação, sem atualização automática de baselines.
- Persistência Flow PostgreSQL permanece como escopo de homologação evolutiva porque não foi possível executar build/runtime local sem SDK.
