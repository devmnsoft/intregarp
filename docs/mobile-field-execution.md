# mobile-field-execution

Documento da Sprint 8 para o MVP Mobile + IA do IntegraRP.

- Mobile é online-first e preparado para fila local futura.
- API mantém isolamento por tenant e RBAC.
- IA é governada, não acessa banco diretamente e executa apenas tools registradas.
- Evidências de campo cobrem foto, arquivo, comentário, GPS, assinatura e POD.
- Fallback humano é criado quando confiança, permissão ou escopo não são suficientes.
