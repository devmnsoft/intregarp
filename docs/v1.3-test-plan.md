# Plano de testes v1.3

1. `dotnet clean`
2. `dotnet restore`
3. `dotnet build`
4. `dotnet test`
5. `cd apps/IntegraRP.Mobile && npm install && npm run typecheck`
6. `docker compose build && docker compose up -d && docker compose ps`
7. Executar `/database/scriptcompleto.sql` duas vezes no mesmo banco.
8. Consultar as views `integrarp.vw_v13_*`.
9. Chamar endpoints `/api/validation/*` com usuário autorizado.
