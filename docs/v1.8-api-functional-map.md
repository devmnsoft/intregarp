# v1.8 — Mapa funcional da API

Padrão: `[Authorize]`, RBAC, tenant obrigatório, `CancellationToken`, `ProblemDetails`, DTOs, use cases e repositories. Controllers não devem conter SQL nem Dapper.

Endpoints de ações: confirmar/cancelar/faturar pedido, assumir/concluir tarefa, processar/reprocessar outbox e mover card de projeto.

Endpoints de validação: módulos, telas, CRUDs, templates, jornadas e dashboard.
