# Script complettop canônico

Base v1.19: `629fb0db7d4db1d34ab86f25dfac9b02268717cc`.

Este documento registra o escopo implementado, comandos esperados e limitações honestas. O PR deve permanecer Draft enquanto PostgreSQL 16 real, Windows standalone, Linux standalone, Flow persistido, CI e evidências remotas não estiverem comprovadamente verdes.

## Comandos principais

```bash
./scripts/generate-script-completop.sh
psql -X "postgresql://integrarp:integrarp@localhost:5432/integrarp_ci" --set ON_ERROR_STOP=1 --file database/script_completop.sql
psql -X "postgresql://integrarp:integrarp@localhost:5432/integrarp_ci" --set ON_ERROR_STOP=1 --file database/validate_script_completop.sql
```

## Limitações

Sem `dotnet`, `psql` e PostgreSQL 16 disponíveis neste ambiente local, as validações reais ficam registradas como pendentes e não são declaradas verdes.
