# IntegraRP v1.16 — Release Notes

## Entregue

- Correção do teste de comentário de tarefa para o contrato `ICurrentExecutionContext`.
- Restrição do proxy Web/BFF por allowlist para reduzir risco de proxy aberto/SSRF.
- Repasse de `Authorization`, `X-Correlation-Id` e `X-Tenant-Id` do Web para a API.
- Swagger restrito ao ambiente Development.
- CORS em Production condicionado a `Cors:AllowedOrigins`.
- Remoção de tokens mobile de demonstração e de `Guid.NewGuid()` como usuário corrente nos controllers mobile.
- Substituição de endpoints mobile no-op por `501 ProblemDetails` honesto quando a persistência ainda não está implementada.

## Riscos restantes

- Login/refresh real persistido ainda precisa ser implementado para Web e Mobile.
- Flow, Project, Studio e BI ainda têm dependências in-memory relevantes.
- Banco, migrations, Docker e testes .NET não puderam ser executados neste container.
