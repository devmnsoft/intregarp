# IntegraRP v1.16 — Jornadas de Usuário

## Disponível parcialmente

- Web consome API via BFF com allowlist e preservação de identidade HTTP.
- Mobile autenticado por Bearer token externo consegue consultar `me`, dashboard, tarefas, evidências, assinatura, conclusão e aprovação usando `ICurrentUserContext`.
