# database-scriptcompleto

O script completo fica em /database/scriptcompleto.sql, é idempotente, usa apenas schema integrarp, pgcrypto, schema_migrations, função fn_set_atualizado_em e objetos v1.2.

## Como testar
Execute a migration 0014 ou o script completo em PostgreSQL sandbox. Use dados demo v1.2 e não configure credenciais reais.
